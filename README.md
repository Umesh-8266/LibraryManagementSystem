# LibraryManagementSystem
Simple Library Management System using ASP.NET Core and SQL, MVC with Ado.net

# Library Management System

## Features
- Add Book
- Edit Book
- Delete Book
- Issue Book
- Return Book
- Search Book

## Technology Used
- ASP.NET Core MVC
- C#
- ADO.NET
- SQL Server

## Database
Table: Books
- Id
- Title
- Author
- TotalCopies
- AvailableCopies

## How to Run
1. Open project in Visual Studio
2. Configure connection string
3. Run project
